In March 2024, I created a command line script in Python to automatically generate a table of contents from existing headings in markdown files. Additionally, the script includes external links, making it a practical tool for use in my own Coding Cookbook. This side project emerged from my growing appreciation for the importance of code documentation and the elegance of well-written markdown files.
