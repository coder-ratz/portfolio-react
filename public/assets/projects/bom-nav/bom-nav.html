In 2023, a project request prompted the extraction of bill of material (BOM) data from Excel. Recognizing the recursive nature of the task, a Python script was initially created to complete the job efficiently. The project evoked my deeper interest in developing a user-friendly web interface using HTML, JavaScript, and Flask.

Main view of the BOM Navigator web application

Screenshot from Faiveley Transport Bochum GmbH

The web tool includes routines to nightly import SQL data from the ERP system, enabling users to navigate BOMs across various levels by clicking item numbers. Additionally, an SQL database compiling historical sales data was integrated into the BOM manager. To address faulty item number inputs, a small API service was added.
